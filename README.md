# NLPs

Natural Language Processing (NLP) Learning Repository

This repository is for learning and experimenting with various NLP techniques and libraries.

## Libraries Used

- **tomotopy**: Topic modeling library
- **matplotlib**: Visualization and plotting
- **wordcloud**: Word cloud generation
- **gensim**: Topic modeling and NLP utilities
- **Base**: Custom base module for common NLP operations

## Installation

Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the sample NLP learning script:

```bash
python nlp_learning.py
```

## Project Structure

- `Base.py`: Base module with common NLP utilities
- `nlp_learning.py`: Sample script demonstrating library usage
- `requirements.txt`: Python dependencies
